Deploy your annotations from [semantify.it](http://semantify.it/) to your WordPress website.

== Description ==

Easy out of the box annotation deployment to your website.


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/semantify-it` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Plugin Name screen to configure the plugin
4. Add your API key.


== Frequently Asked Questions ==

= Do I need an account in semantify.it? =

Yes, without semantify.it this plugin will not work.
